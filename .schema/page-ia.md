## 0. 우선순위 범례

- 🔴 **Phase 1 (MVP 핵심)** — 클립 생성·타임라인 워크플로우 검증에 필수
- 🟡 **Phase 1-B** — 창작 전처리 파이프라인 (가사→시나리오→스토리보드→첫장면)
- 🔵 **Phase 2** — 엔진 선택 / 외부 API 연동
- ⚪ **Phase 3+** — 소셜 공유, 유료 전환, 개인화 (지금 디자인 우선순위 낮음)

## 1. 공통 레이아웃 셸 (Global Shell)

모든 로그인 이후 페이지가 공유하는 뼈대. 페이지별 디자인 생성 시 이 셸을 기준으로 콘텐츠 영역만 교체합니다.

- **Top Nav (고정, 높이 ~64px)**
  - 좌측: 서비스 로고 + 현재 프로젝트명(있을 경우)
  - 중앙/우측: 홈 · 내 프로젝트 · 설정 진입 링크
  - 우측 끝: 사용자 아바타(드롭다운 → 설정/로그아웃)
- **프로젝트 작업 중 서브 네비게이션 (탭 형태, 프로젝트 컨텍스트 내에서만 노출)**
  - 컨셉 설정 → (선택)창작 전처리 → 클립 생성/검토 → 타임라인 편집 → 출력/공유
  - 현재 단계 하이라이트, 이전 단계로 자유 이동 가능(단, 이후 단계는 데이터 존재 시에만 활성화)
- **전역 상태 요소**: 저장 상태 인디케이터(자동저장), 생성 작업 진행 중일 때 우측 상단 미니 진행 배지

## 3. 페이지별 상세

### P01. 온보딩/로그인
- **목적**: 서비스 첫 진입, 소셜 로그인 유도
- **정보 구조**: 서비스 가치 제안 헤드라인 1개 + 핵심 기능 3가지 요약 카드 + Google/카카오 로그인 버튼 2개
- **핵심 컴포넌트**: 로고, 히어로 카피, 소셜 로그인 버튼(2), 하단 약관/개인정보 링크
- **상태**: 기본 / 로그인 처리 중(스피너) / 로그인 실패(에러 토스트)
- **다음 이동**: 로그인 성공 → P02 홈 대시보드 (최초 로그인 시 프로필 확인 모달 노출 가능)

### P02. 홈 대시보드
- **목적**: 로그인 후 첫 진입점, 새 프로젝트 시작 또는 기존 작업 재개
- **정보 구조**: "새 프로젝트 만들기" 대형 CTA 카드 / 최근 프로젝트 3~4개 미리보기(썸네일+진행상태) / 추천 스타일 탐색 배너
- **핵심 컴포넌트**: 프로젝트 카드(썸네일, 제목, 마지막 수정일, 진행 단계 배지), 새 프로젝트 CTA
- **상태**: 프로젝트 없음(빈 상태 일러스트+CTA 강조) / 프로젝트 존재
- **다음 이동**: 새 프로젝트 → P05 / 기존 프로젝트 클릭 → 해당 프로젝트의 마지막 작업 단계로 복귀 / "전체보기" → P03

### P03. 내 프로젝트 목록
- **목적**: 모든 프로젝트를 조회·관리
- **정보 구조**: 그리드/리스트 뷰 전환, 정렬(최신순/이름순), 프로젝트별 진행 단계 배지, 삭제 액션
- **핵심 컴포넌트**: 프로젝트 카드 그리드, 검색/필터 바, 삭제 확인 모달
- **상태**: 빈 목록 / 목록 있음 / 삭제 확인 다이얼로그
- **다음 이동**: 카드 클릭 → 프로젝트 재진입, 삭제 → 목록 갱신

### P04. 추천 스타일 탐색 *(Phase 3+, 디자인 우선순위 낮음)*
- **목적**: 영감 제공용 스타일 프리셋 갤러리
- **정보 구조**: 스타일 프리셋 카드 그리드(썸네일 영상/이미지 + 스타일명), 필터(장르별 테마)
- **다음 이동**: 프리셋 선택 → 새 프로젝트 생성으로 이어지며 P06에 사전 반영

### P05. 음악 업로드 & AI 컨셉 설정
- **목적**: 프로젝트 시작점 — 음악 업로드 + 컨셉 입력(시나리오 A/B 분기점)
- **정보 구조**:
  1. 음악 파일 업로드 영역 (드래그앤드롭 + 파일 선택), 업로드 후 파형 미리보기 + 재생 컨트롤
  2. AI 음악 분석 결과 카드(장르/템포/분위기 자동 제안) — 분석 완료 후 노출
  3. 컨셉 입력 탭: [텍스트 프롬프트] / [이미지 레퍼런스 업로드] / [키워드 선택]
  4. 하단 분기 CTA: "빠른 생성으로 진행"(시나리오B → P06) vs "가사부터 만들기"(시나리오A → P08)
- **상태**: 업로드 전 / 분석 중(로딩) / 분석 완료
- **다음 이동**: P06 또는 P08

### P06. 비주얼 스타일 선택
- **목적**: 영상 생성 전 스타일/톤 확정 (시나리오 B 경로)
- **정보 구조**: 스타일 프리셋 카드 그리드(애니메이션/실사/추상/흑백 등), 장르별 테마 프리셋, 선택 시 우측 요약 패널(현재까지 설정값)
- **핵심 컴포넌트**: 프리셋 카드(호버 시 미리보기), 선택 요약 사이드바, "생성 시작" CTA
- **다음 이동**: 생성 시작 → P07

### P07. AI 영상 생성 중
- **목적**: 클립 생성 대기 상태 안내 (여러 클립 병렬/순차 생성 가능)
- **정보 구조**: 진행률 바 + 예상 소요 시간, 생성 중인 클립 개수/완료 개수 카운터, 취소 옵션
- **상태**: 생성 중 / 일부 완료 / 전체 완료 / 오류(재시도 CTA)
- **다음 이동**: 완료 → P12 클립 검토로 자동 전환

### P08. 가사 작사 도우미 *(🟡 Phase 1-B)*
- **목적**: 키워드·감정·주제 입력 → AI 가사 초안 생성 및 편집
- **정보 구조**:
  1. 좌측 입력 패널: 키워드/감정/주제 입력, 언어 선택(한/영/혼용), 스타일 프리셋(서정적/랩플로우 등)
  2. 우측 편집 영역: 절/후렴/브릿지 블록형 에디터, 블록별 음절 수 표시
  3. 버전 히스토리 사이드 패널(이전 생성본 비교)
- **핵심 컴포넌트**: 블록 에디터, AI "다시 생성"/"이 부분만 개선" 버튼, 버전 탭
- **다음 이동**: 가사 확정 → P09 (시나리오 생성)

### P09. 시나리오(씬) 생성 *(🟡 Phase 1-B)*
- **목적**: 확정 가사 기반 씬 구조 자동 생성 및 편집
- **정보 구조**: 씬 카드 리스트(세로 배열, 절/후렴/브릿지 구간과 매핑), 각 카드에 배경/분위기/카메라무드/예상 길이 필드, 상단에 전체 흐름 요약 스트립
- **핵심 컴포넌트**: 씬 카드(드래그 정렬 가능), 카드 인라인 편집, 씬 추가/삭제 버튼
- **다음 이동**: 시나리오 확정 → P10

### P10. 스토리보드 스틸컷 *(🟡 Phase 1-B)*
- **목적**: 씬별 스틸컷 이미지 생성 및 스타일 일관성 확인
- **정보 구조**: 그리드 뷰(씬 순서대로 스틸컷 썸네일), 씬 선택 시 상세 패널(프롬프트 확인/수정, 재생성, 버전 비교, 외부 이미지 업로드 대체)
- **핵심 컴포넌트**: 스토리보드 그리드, 상세 편집 사이드패널, 버전 비교 뷰(A/B)
- **다음 이동**: 스틸컷 확정 → P11 (씬 단위 or 전체 일괄 전송)

### P11. 첫장면 클립 생성 *(🟡 Phase 1-B)*
- **목적**: 확정 스틸컷을 첫 프레임으로 하는 영상 클립 생성
- **정보 구조**: 씬별 카드(스틸컷 썸네일 + 움직임 방향/강도 설정 슬라이더 + 클립 길이) + 전체 일괄 생성 옵션
- **핵심 컴포넌트**: 움직임 설정 컨트롤(패닝/틸팅/줌, 강도), 일괄 생성 진행 표시
- **다음 이동**: 클립 생성 완료 → P12 (타임라인에 자동 배치된 상태로 진입)

### P12. 클립 검토 & 채택
- **목적**: 생성된 클립을 검토하여 채택/기각/재생성 (전체 워크플로우의 핵심 반복 루프)
- **정보 구조**: 클립 썸네일 목록(호버 시 자동재생), 각 클립에 채택/기각/재생성 액션 버튼, 재생성 시 동일 구간 A/B 비교 뷰
- **핵심 컴포넌트**: 클립 카드, 비교 뷰 모달, 재생성 사유/프롬프트 수정 입력
- **다음 이동**: 채택 → 타임라인에 추가되며 P13으로 이동 가능 (모든 구간 채택 전까지 이 화면 반복 방문)

### P13. 타임라인 편집
- **목적**: 채택된 클립을 배치·조합하여 전체 뮤직비디오 완성 (플랫폼의 핵심 편집기)
- **정보 구조**:
  1. 상단: 전체 미리보기 플레이어
  2. 중단: 타임라인 트랙 — 음악 파형 + 구간 마커(절/후렴/브릿지) + 클립 블록(드래그로 순서 변경/트리밍)
  3. 좌측 또는 하단: 클립 라이브러리(미채택 클립, 재생성 요청 가능)
  4. 툴바: Undo/Redo, 전환 효과 설정, 빈 구간 경고 인디케이터
- **핵심 컴포넌트**: 파형 렌더러, 드래그 가능한 클립 블록, 트리밍 핸들, 전환효과 팝오버, Undo/Redo 버튼
- **상태**: 빈 구간 있음(경고 하이라이트) / 전체 채움 완료
- **다음 이동**: "편집 완료" → P15 (또는 텍스트/효과 편집이 필요하면 P14)

### P14. 텍스트/효과 편집 *(⚪ Phase 3+)*
- **목적**: 텍스트 오버레이, 색상 필터 등 부가 편집
- **정보 구조**: 미리보기 캔버스 + 우측 속성 패널(텍스트 폰트/크기/색상/위치, 필터 강도 슬라이더)
- **다음 이동**: 저장 → P13으로 복귀 또는 P15로 진행

### P15. 최종 미리보기 & 다운로드
- **목적**: 완성작 확인 및 출력
- **정보 구조**: 전체 재생 플레이어, 해상도/비트레이트 선택, 워터마크 제거 옵션(유료 배지), 다운로드 버튼
- **상태**: 렌더링 중(진행률+예상시간) / 렌더링 완료 / 렌더링 오류
- **다음 이동**: 다운로드 완료 → P16(공유) 또는 P02(홈)로 복귀

### P16. 소셜 미디어 공유 *(⚪ Phase 3+)*
- **목적**: YouTube/Instagram/TikTok 공유
- **정보 구조**: 플랫폼 선택 카드, AI 자동 생성 캡션/해시태그 미리보기(수정 가능), 공유 이력 리스트

### P17. 설정 홈 *(🔵 Phase 2)*
- **목적**: 계정/엔진/API 설정 진입 허브
- **정보 구조**: 설정 카테고리 리스트(계정, 생성 엔진, 외부 API 연동)
- **다음 이동**: 각 카테고리 → P18 / P19

### P18. 생성 엔진 선택 *(🔵 Phase 2)*
- **목적**: 기본 엔진(seedance 2.0) 및 대안 엔진(Kling 3.0 등) 선택
