# AI 기반 뮤직 비디오 제작 플랫폼 제품 기획서 (PRD)

## 1. 개요
* **한 줄 정의:** AI 기반 뮤직 비디오 제작 플랫폼
* **제품 목표:** 개인 창작자 및 취미 활동가들이 높은 제작 비용과 전문 기술의 장벽 없이, 쉽고 빠르게 고품질의 AI 뮤직 비디오를 제작할 수 있도록 지원하여 뮤직 비디오 제작의 민주화를 달성합니다.
* **배경:** 최근 AI 기술의 발전은 콘텐츠 제작 분야에 혁신적인 변화를 가져오고 있습니다. 특히 개인 음악 창작자와 아마추어 뮤지션들은 자신의 음악을 시각적으로 표현하고 싶어 하지만, 높은 뮤직 비디오 제작 비용과 전문 영상 제작 기술의 부재로 어려움을 겪고 있습니다. 이러한 배경 속에서 AI 기술을 활용하여 누구나 쉽게 뮤직 비디오를 제작할 수 있는 플랫폼에 대한 니즈가 증가하고 있습니다.

## 1-A. 프로젝트 스코프 및 확장 방향

### 현재 스코프 (비상업적 프로토타입)
> 이 프로젝트는 현재 상업화를 목표로 하지 않습니다.
> 개인 창작자를 위한 AI 뮤직 비디오 제작 워크플로우를 검증하고, 향후 확장 가능한 구조와 UI를 미리 갖추는 것이 목표입니다.

* **현재 단계:** 기능 구조 및 UI/UX 설계, 핵심 워크플로우 검증
* **대상 사용자:** 소규모 개인 창작자, 인디 아티스트, 아마추어 음악 프로듀서

### 확장성 설계 원칙
향후 확장을 고려하여 다음 원칙을 설계 단계부터 반영합니다.

| 원칙 | 내용 |
|---|---|
| **창작 전처리 파이프라인** | 가사 작사 → 시나리오 → 스토리보드 스틸컷 → 첫장면 클립 생성으로 이어지는 순차적 창작 보조 흐름 |
| **오픈 백엔드** | 특정 생성 엔진에 종속되지 않고, 엔진 선택 및 교체가 가능한 구조 |
| **API 키 기반 연동** | 사용자가 직접 외부 API 키를 입력하여 Atlas Cloud 등 외부 서비스를 연결할 수 있는 구조 |
| **클립 단위 워크플로우** | 긴 영상을 한 번에 생성하는 대신, 10초 내외 클립의 반복 생성·검토·선택·재생성 방식으로 품질 수렴 |
| **타임라인 편집** | 채택된 클립을 타임라인에 조합·재배열하여 전체 뮤직 비디오를 구성하는 전용 편집 인터페이스 |
| **유료 기능 구조 준비** | 워터마크 제거, 고해상도 렌더링 등 유료 전환 가능한 기능의 UI 구조를 미리 설계 |

### 향후 확장 로드맵 (예비)
```
Phase 1 (현재) : 핵심 워크플로우 검증 / 클립 생성·타임라인 편집 UI 구현
Phase 1-B      : 창작 전처리 파이프라인 구현
                 (가사 작사 도우미 → 시나리오 생성 → 스토리보드 스틸컷 → 첫장면 클립 생성)
Phase 2         : 생성 엔진 선택 UI / 외부 API 키 연동 기능 구현
Phase 3         : 소셜 공유 / 유료 플랜 구조 도입
Phase 4         : 커뮤니티 / 피드백 기반 개인화 추천 고도화
```

## 2. 문제 및 해결 방안

### 2.1 사용자 문제
* **비용 및 기술 장벽:** 개인 창작자들은 뮤직 비디오 제작 시 높은 비용과 전문적인 영상 편집 기술의 부재로 인해 자신의 음악을 시각적으로 표현하는 데 큰 어려움을 겪고 있습니다.
* **생성 제어의 한계:** 긴 영상을 한 번에 만드는 생성 방식은 창작자의 의도를 정교하게 반영하기 어렵기 때문에, 짧은 클립을 빠르게 반복 생성·검토·선택하며 원하는 결과에 수렴하는 작업 방식이 필요합니다.

### 2.2 해결 방안
* **클립 기반 타임라인 워크플로우:** 사용자가 음악과 레퍼런스(이미지/동영상/음악 구간)를 기반으로 10초 내외의 영상 클립을 반복 생성하고, 채택한 클립을 **타임라인에 드래그하여 조합**해 전체 뮤직 비디오를 완성할 수 있는 AI 제작 워크플로우를 제공합니다.
* **유연한 생성 엔진 및 인프라 연동:** 기본 동영상 생성 엔진은 `seedance 2.0`으로 하되, 사용자가 `Kling 3.0` 등 다른 엔진을 선택할 수 있으며, `Atlas Cloud` 같은 외부 이미지·동영상 API 플랫폼은 사용자가 개인 API 키를 입력해 직접 연결할 수 있도록 지원합니다.

### 2.3 차별점
* **점진적 품질 향상 (Iterative Refinement):** 10초 내외 클립 단위의 빠른 반복(생성 → 검토 → 선택 → 재생성) 프로세스로 최종 결과물의 품질을 극대화합니다.
* **오픈 백엔드 아키텍처:** 특정 엔진에 종속되지 않고 엔진 선택(seedance 2.0 기본) 및 외부 API 키 기반 연결(Atlas Cloud 등)을 지원하여, 사용자가 자신의 워크플로우와 비용/품질 선호에 맞춰 생성 백엔드를 유연하게 구성할 수 있습니다.
* **타임라인 기반 조합 편집:** 생성된 클립을 타임라인 위에서 시각적으로 배치·순서 변경·길이 조절하여 전체 뮤직 비디오를 구성하는 전용 인터페이스를 제공합니다.

## 3. 타겟 및 시나리오

### 3.1 타겟 사용자
* **대상:** 음악을 취미로 하는 개인, 인디 아티스트, 아마추어 음악 프로듀서, 유튜버 등 자신의 음악을 시각화하여 더 많은 사람들과 공유하고자 하는 모든 개인 창작자.
* **특징:** 전문 영상 제작 지식이나 고가의 장비 없이도 자신의 음악과 어울리는 매력적인 뮤직 비디오를 직관적으로 만들고 싶어 하는 니즈를 가짐.

### 3.2 사용자 시나리오

#### 시나리오 A — 창작 전처리 파이프라인 경로 (풀 워크플로우)
1. **음원 업로드 및 컨셉 설정:** 자신의 음악 파일을 업로드하고, AI 디렉터와 협업하여 비주얼 컨셉(분위기, 스타일, 키워드)을 설정합니다.
2. **가사 작사 도우미:** 키워드·감정·주제를 입력하면 AI가 음악에 맞는 가사 초안을 생성하고, 사용자가 절/후렴/브릿지 단위로 편집·확정합니다.
3. **뮤직 비디오 시나리오 생성:** 확정된 가사를 기반으로 AI가 씬 구조(배경, 분위기, 카메라 무드)를 담은 시나리오를 자동 생성하고, 사용자가 씬을 수정·재배열합니다.
4. **스토리보드 스틸컷 생성:** 각 씬의 설정을 기반으로 AI가 스틸컷 이미지를 생성하며, 사용자는 씬별로 재생성·비교하여 확정합니다.
5. **첫장면 기반 클립 생성:** 확정된 스틸컷을 첫 프레임으로 하여 AI가 10초 내외의 영상 클립을 생성합니다.
6. **타임라인 조합 편집:** 생성된 클립을 타임라인에 배치하고, 순서·길이·전환 효과를 조절하여 전체 뮤직 비디오를 완성합니다.
7. **최종 출력 및 공유:** 완성된 뮤직 비디오를 다운로드하거나 소셜 미디어로 공유합니다.

#### 시나리오 B — 빠른 생성 경로 (컨셉 직접 입력)
1. **음원 업로드 및 컨셉 설정:** 음악을 업로드하고 텍스트 프롬프트·이미지 레퍼런스로 컨셉을 직접 설정합니다.
2. **생성 엔진 선택(선택사항):** 기본 엔진(seedance 2.0) 또는 원하는 엔진/외부 API를 선택합니다.
3. **AI 기반 클립 생성 및 검토:** AI가 컨셉에 맞는 클립을 생성하고, 사용자가 채택·기각·재생성을 반복합니다.
4. **타임라인 조합 편집 → 최종 출력:** 클립을 조합하여 뮤직 비디오를 완성하고 공유합니다.

## 4. 성공 및 위험 요소

### 4.1 핵심 지표 (KPI)
* **성장성:** 월간 활성 사용자 수 (MAU)
* **활성화:** 서비스 내에서 생성 및 최종 빌드된 뮤직 비디오 수
* **수익성(확장 시):** 유료 구독 및 프리미엄 플랜 전환율
* **만족도:** 사용자 만족도 조사 점수 및 정성적 피드백
* **확산성:** 서비스 내에서 제작되어 유튜브, 틱톡 등 외부 소셜 미디어에 공유된 영상의 누적 조회수

### 4.2 리스크 및 대응 방향
* **법적 리스크:** AI 모델의 저작권 문제 및 생성된 콘텐츠에 대한 법적 책임 소지
* **품질 리스크:** AI 모델 특성상 발생할 수 있는 생성 영상 콘텐츠의 일관성(Consistency) 부족 문제
* **경쟁 리스크:** 글로벌 빅테크 및 스타트업의 급격한 AI 기술 발전에 따른 유사 경쟁 서비스의 출현
* **보안 리스크:** 사용자 업로드 음원 및 개인 정보 보호, 외부 API 키 저장에 따른 데이터 보안 문제
